## 🎯 Comment démarrer maintenant

### Étape 1: Charger l'extension
```bash
1. chrome://extensions/
2. Activer "Mode développeur" (top-right)
3. "Charger l'extension non empaquetée"
4. Sélectionner le dossier /extension-revesta
```

### Étape 2: Tester
```bash
1. Ouvrir une annonce LeBonCoin

3. Vérifier popup affiche aides
```

### Étape 3: Vérifier logs (F12)
```bash
F12 → Console
Chercher logs ✅ de succès
```
